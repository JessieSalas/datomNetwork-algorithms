Algorithms are essential to understand

This is an informational page about algorithms.

Posted by Datom Blog Network

In mathematics and computer science, an algorithm is a self-contained step-by-step set of operations to be performed. Algorithms exist that perform calculation, data processing, and automated reasoning.

Algorithms have many applications in computer science. Any time there is a problem that can be broken down into a sequence of steps that must be solved, the abstract concept of those steps is the algorithm. For example, one kind of algorithm is a sorting algorithm. Sorting algorithms take the problem of sorted some sort of comparable objects and breaks it dwn to a step-by-step sequence that is guaranteed to solve it.

One sorting algorithm is bubble sort. It is one of the slowest possible sorting algorithms, and there are many alternatives that are faster. This brings us to the idea of runtime. Runtime is the concept of how long the algorithm takes to complete, in terms of an abstract idea of time steps.

We measure the runtime of an algorithms in terms of functions oftentimes. For intance, when a sorting algorithm like bubble sort for example gets passed in a list of N numbers, it takes N^2 time steps to complete, as a function of that input.

Algorithms have applications in other fields besides computer science, however. In modern agriculture, farmers use algorithms for applications like crop-rotation and when to water which plants.
